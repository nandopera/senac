Lista de Símbolos do HTML

  • < - início de uma tag HTML
  • > - fim de uma tag HTML
  • / - fecha uma tag HTML
  • = - atribui um valor a um atributo HTML
  • " - define um valor de atributo dentro das aspas duplas
  • ' - define um valor de atributo dentro das aspas simples
  • <!-- - início de um comentário HTML
  • --> - fim de um comentário HTML
  • & - início de uma entidade HTML
  • ; - fim de uma entidade HTML

Lista de Símbolos do CSS

  • : - separa a propriedade do valor no CSS
  • ; - separa as declarações no CSS
  • { - início de um bloco de declarações no CSS
  • } - fim de um bloco de declarações no CSS
  • / - início de um comentário no CSS
  • * - se usado como um seletor, significa "selecionar todos os elementos"
  • . - se usado como um seletor, significa "selecionar um elemento com uma classe específica"
  • # - se usado como um seletor, significa "selecionar um elemento com um ID específico"

Lista de Símbolos do JavaScript

  • ; - separa as declarações no JavaScript
  • { - início de um bloco de código no JavaScript
  • } - fim de um bloco de código no JavaScript
  • // - início de um comentário em linha no JavaScript
  • /\* e \*/ - início e fim de um comentário em bloco no JavaScript
  • () - utilizados para definir parâmetros de uma função no JavaScript
  • [] - utilizados para definir uma matriz ou acessar seus elementos no JavaScript
  • == e != - operadores de comparação para igualdade ou diferença de valor no JavaScript
  • === e !== - operadores de comparação para igualdade ou diferença de valor e tipo no JavaScript
  • && e || - operadores lógicos "E" e "OU" no JavaScript
  • + - operador de adição no JavaScript
  • - - operador de subtração no JavaScript
  • * - operador de multiplicação no JavaScript
  • / - operador de divisão no JavaScript
  • % - operador de módulo no JavaScript
